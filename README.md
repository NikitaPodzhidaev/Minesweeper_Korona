### Задача по реализации игры "Сапёр". 

#### Игра является десктопной, не имеющей под собой работу с БД и интернетом (хотя архитектура построена таким образом, что добавить разные реализации - несложно)
#### Использованные технологии: Java, Swing, Gradle, архитектура Model-View-Controller, алгоритмы поиска в ширину.
#### Для запуска используется обычная сборка через Gradle, используя, например, Intellij Idea
